## Examples

Operators
```
js          qcq
------------------
x = 5       x is 5
x + 5       x + 5
x - 5       x - 5 
x * 5       x * 5
x / 5       x / 5
x ** 5      x ^ 5
x % 7       x % 7
```

Comparators
```
js          qcq
------------------
x && y      x and y
x || y      x or y
x < y       x < y
x > y       x > y
x <= y      x <= y
x >= y      x >= y
x === y     x == y
x !== y     x != y
```

Conditionals
```
if x == 0
    output “False”
else if x == 1
    output “True”
else
    output "Maybe?"
```

For Loop
```
loop i from 0 to 10 by 1
     output i
end
```

While Loop
```
loop until i != 10
     output 5 * i
end
```

Hello World
```
output "hello world"
```
